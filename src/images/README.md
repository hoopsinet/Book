# Dossier Images

Ce dossier contient toutes les images utilisées dans l'application.

Types d'images acceptés :
- .jpg / .jpeg
- .png
- .svg
- .gif

Structure recommandée :
```
/images
  /book-covers     # Couvertures de livres
  /backgrounds     # Arrière-plans
  /icons          # Icônes
  /illustrations  # Illustrations
```
